Overview

Online Proctored Exam is a platform for conducting secure remote exams with dual-camera proctoring (front + screen share), AI-powered behavior detection, cloud recording, and real-time proctor oversight. Built for universities, certification bodies, and corporate training.

Why Tencent RTC? TRTC's multi-stream publishing (camera + screen share simultaneously) powers dual-camera proctoring. Server-side recording captures everything for post-exam review. Low-latency conferencing lets proctors intervene in real time.

Key features

  • Dual-camera proctoring — front camera + screen share streamed simultaneously.
  • AI behavior detection — face absence, multiple faces, eye tracking anomaly, phone detection.
  • Real-time proctor dashboard — grid view of all examinees, flag alerts, intervene via voice.
  • Cloud recording — full session recording for post-exam audit.
  • Lockdown browser — prevents tab switching, copy/paste, external app access.
  • Automated flagging — suspicious events timestamped and logged for review.
